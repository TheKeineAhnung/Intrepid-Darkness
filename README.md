## Installation Guide

1. Open Extensions sidebar panel in Visual Studio Code.
2. Search for Intrepid Darkness.
3. Click Install.
4. Click Reload.
5. Code/File > Preferences ＞ Color Theme ＞ Intrepid Darkness.
6. Rate the theme.
